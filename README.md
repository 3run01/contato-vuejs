# app

## Configuração do projeto
```
yarn install
```

### Usar em modo de desenvolvimento
```
yarn serve
```

### Compilar para modo de produção
```
yarn build
```

### Deploy produção

Ao compilar para modo de podução será criada uma pasta chamda dist, do domínio deverá apontar a mesma usando NGINX.
